工具函数¶

主要是快速排序函数和字符串拼接函数。

使用 operator+ 合并多个字符串时，可能引起内存的反复移动，导致效率低下。

一种解决方法是，合并前先在 string 里预留空间（reserve），之后把每个字符串都附加到 string 之后（operator+=）。

代码实现参考了 https://stackoverflow.com/a/18899027。

template<class Iter, class Compare = std::less<typename Iter::value_type>>
void Sort(Iter first, Iter last, Compare cmp)¶

Sort the elements of a sequence using a predicate for comparison.

参数

  • first – An iterator.

  • last – Another iterator.

  • cmp – A comparison functor.

返回

Nothing. Sorts the elements in the range [first, last) in ascending order, such that cmp(*(i+1),*i) is false for every iterator i in the range [first,last-1). The relative ordering of equivalent elements is not preserved.
