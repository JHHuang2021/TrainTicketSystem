B+ 树¶

template<class Key, class Value, int kInternalSize = 400, int kLeafSize = 10, int kInternalBufferSize = 400, int kLeafBufferSize = 400>
class huang::BPlusTree¶

Implementation of simple B+ tree data structure where internal pages direct the search and leaf pages contain actual data.

  • We only support unique key.

  • Support insert & remove.

  • The structure should shrink and grow dynamically.
